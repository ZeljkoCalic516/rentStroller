# rentStroller

Baby gear rental site (Belgrade). Static HTML/CSS, local dev: `npm run dev` → http://localhost:3333

## Product images (`images/`)

Drop one photo per listing. Use these **exact filenames** (JPG or PNG bytes are fine; keep the name):

| File | Use |
|------|-----|
| `logo.png` | Header logo (left of “Rent Stroller”) |
| `hero.jpg` | Home page hero banner |
| `stroller1.jpg` | City stroller |
| `stroller2.jpg` | Travel stroller |
| `high-chair.jpg` | High chair (feeding chair) |
| `car-seat.jpg` | Car seat |
| `toddler-scooter.jpg` | Toddler push scooter |

Rename in Finder only changes the extension label—export/save as JPEG or PNG with the name above, or use Preview **Export** if converting format.
